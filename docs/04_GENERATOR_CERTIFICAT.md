## CALCULELE DE BAZĂ

### 1. CIFRA VIEȚII (Life Path Number)

```
Formula: Suma tuturor cifrelor din data nașterii, redusă la 1-9 (sau 11, 22, 33)

Exemplu: 21.07.1981
→ 2+1+0+7+1+9+8+1 = 29
→ 2+9 = 11 ← NUMĂR MAESTRU (nu se mai reduce!)

Alte exemple:
→ 15.03.1990: 1+5+0+3+1+9+9+0 = 28 → 2+8 = 10 → 1+0 = 1
→ 08.11.1985: 0+8+1+1+1+9+8+5 = 33 ← NUMĂR MAESTRU!
→ 22.04.2001: 2+2+0+4+2+0+0+1 = 11 ← NUMĂR MAESTRU!

Numerele Maestru: 11, 22, 33 — NU se reduc mai departe
```

### 2. CIFRA ZILEI (Talent Nativ)
```
Formula: Ziua nașterii redusă la o singură cifră
Exemplu: ziua 21 → 2+1 = 3
         ziua 29 → 2+9 = 11 (Maestru)
         ziua 11 → rămâne 11 (Maestru)
```

### 3. CIFRA LUNII (Lecția Emoțională)
```
Formula: Luna redusă
Exemplu: luna 7 → 7
         luna 11 → 11 (Maestru)
         luna 12 → 1+2 = 3
```

### 4. CIFRA ANULUI NAȘTERII (Contextul Cosmic)
```
Formula: Suma cifrelor anului
Exemplu: 1981 → 1+9+8+1 = 19 → 1+9 = 10 → 1+0 = 1
         1990 → 1+9+9+0 = 19 → 10 → 1
         2001 → 2+0+0+1 = 3
```

### 5. CIFRA CONTRACTULUI SOCIAL (din CNP)
```
Formula: Suma TUTUROR cifrelor din CNP, redusă

CNP România: S AA LL ZZ JJ NNN C
Exemplu CNP: 1810721xxxxxx
→ Suma toate 13 cifre → reducere la 1-9 (sau 11, 22, 33)

Semnificație: Ce "vrea" societatea/sistemul de la tine
```

### 6. TENSIUNEA DIVINĂ
```
Formula: Cifra Vieții vs Cifra Contractului Social

ARMONIE (aceleași numere sau complementare):
→ Viața ta naturală COINCIDE cu ce cere societatea
→ Drumul tău este mai lin, misiunile se aliniază

TENSIUNE (numere opuse sau diferite):
→ Există un conflict intern între natura ta și așteptările sociale
→ Aceasta este sursa principalelor lecții ale vieții tale
→ Rezolvarea acestei tensiuni = ascensiunea ta principală
```
