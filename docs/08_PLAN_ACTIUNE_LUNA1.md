## SĂPTĂMÂNA 3 (15-21 Aprilie 2026)
### TEMA: PLATFORMA DIGITALĂ MVP + CONȚINUT

**MVP = Minimum Viable Product**
Scopul: Să poți genera un certificat funcțional până la finalul săptămânii

**Opțiunea A — Rapid (recomandat pentru lansare):**
Un Google Form / Typeform care:
1. Primește data nașterii + codul fiscal/CNP
2. Calculează automat cifrele (Google Sheets cu formule)
3. Generează un PDF personalizat
4. Îl trimite pe email

Cost: 0-50 EUR | Timp implementare: 2-3 zile

**Opțiunea B — Profesional (pentru lansare oficială):**
Site web cu formular integrat și generare PDF automată
Cost: 500-2000 EUR | Timp implementare: 2-4 săptămâni

**Acțiuni săptămâna 3:**
- [ ] Construcția MVP (Opțiunea A) — testabil intern
- [ ] Generarea a 10 certificate test pentru validare
- [ ] Redactarea primelor 5 postări pentru rețelele sociale
- [ ] Filmarea unui video de prezentare scurt (60-90 secunde)
  - Conținut: "Ce este Certificatul Numerologic al Vieții?"
  - Format: Vorbire directă la cameră, pe fond simplu, luminos
- [ ] Scrierea primului articol de blog/presă (500-800 cuvinte)

**Conținut social media săptămâna 3:**
- Post 1: "De ce a venit copilul tău pe lume?" (întrebare cu impact)
- Post 2: Infografic — "Cele 5 cifre care îți definesc viața"
- Post 3: Demo — "Cum arată Certificatul Numerologic al Vieții"
- Post 4: Citat din Manifest + vizual
- Post 5: "Povestea din spatele proiectului" (fondatorul se prezintă)

## SĂPTĂMÂNA 4 (22-30 Aprilie 2026)
### TEMA: LANSAREA PUBLICĂ + PRIMELE CONTACTE INSTITUȚIONALE

**EVENIMENTUL DE LANSARE**

Data propusă: **30 Aprilie 2026** (Ziua Internațională a Jazz-ului — energie creativă globală)
Alternativă: **1 Mai 2026** (Ziua Internațională a Muncii — drepturile universale)

Format recomandat: **Online Webinar** (accesibil global)
Platformă: Zoom / YouTube Live / Facebook Live
Durata: 90 minute
Structură:
- 0-10 min: Deschidere + muzică + introducere
- 10-30 min: Prezentarea Manifestului Global (lectură + comentariu)
- 30-50 min: Demo live — generarea unui certificat în timp real
- 50-70 min: Prezentarea certificatului fondatorului (personal, emotional)
- 70-85 min: Q&A
- 85-90 min: Cum te poți alătura + CTA

**Invitați la lansare:**
- Numerologi cunoscuți din România
- Psihologi și terapeuți din comunitate
- Jurnaliști și bloggeri de spiritualitate / dezvoltare personală
- Reprezentanți ai asociațiilor de părinți
- Orice persoane publice simpatizante

**Primele contacte instituționale (Săptămâna 4):**
