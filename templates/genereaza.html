Întrebări frecvente

Este sigur să introduc CNP-ul? +
Da. CNP-ul este procesat local, pe serverul tău, doar pentru calculul cifrelor numerologice. Nu este stocat în nicio bază de date și nu este transmis nicăieri.
Ce este Cifra Vieții? +
Cifra Vieții (Life Path Number) este cea mai importantă cifră numerologică. Se calculează prin reducerea tuturor cifrelor din data nașterii la o singură cifră (sau număr maestru 11, 22, 33). Reprezintă misiunea ta fundamentală în această viață.
Ce sunt numerele maestru? +
Numerele 11, 22 și 33 sunt considerate "numere maestru" în numerologie — configurații rare cu o misiune spirituală specială. Dacă ai un număr maestru, certificatul tău va evidenția acest lucru.
Pot genera certificatul pentru altcineva? +
Da! Poți genera certificate pentru oricine — copii, partener, prieteni. Ai nevoie doar de data nașterii și CNP-ul lor.
